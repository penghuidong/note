# mysql表数据检索

## select语句
### 检索单个列
	select 字段名 from 表名;
### 检索多个列
	select 字段名1, 字段名2, 字段名3 from 表名;
### 检索所有列
	select * from 表名;
### 返回不同的值
	select distinct 字段名 from 表名;//使用distinct关键字，必须放在列名的前面，不能部分使用distinct
### 限制结果
    // 限制返回不多于5行
    select 字段名 from 表名 limit 5;
    // 限制返回从第5行开始的5行
    select 字段名 form 表名 limit 5,5;

__在行数不够时，mysql只返回它能返回的那么多行__

### 使用完全限定的表名
	select 表名.字段名 from 表名;
